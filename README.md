# Smart Feedback Analysis System

This project analyzes user feedback using NLP and Machine Learning.

## Features
- Sentiment analysis (Positive, Negative, Neutral)
- Text preprocessing
- Data visualization
- Streamlit dashboard

## Technologies Used
- Python
- Pandas
- NLTK / TextBlob
- Streamlit
- Matplotlib

## How to Run
1. Install requirements  
   pip install -r requirements.txt
2. Run the app  
   streamlit run app.py
